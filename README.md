# Scalable Capital — ownCloud app

Multi-user ownCloud 10 port of [`Scalable-Capital-Dashboard`](../Scalable-Capital-Dashboard/).
Each ownCloud user can connect their own Scalable Capital account
(Broker + Wealth) and view their portfolio inside the ownCloud UI.

Powered upstream by [`sc-api`](../sc-api/).
